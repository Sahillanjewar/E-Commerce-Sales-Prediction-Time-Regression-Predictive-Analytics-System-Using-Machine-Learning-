🚀 Project Overview

This project develops a Predictive Analytics System that forecasts E-Commerce sales using Time Series Regression and Machine Learning algorithms.

The system helps businesses:

Forecast future sales

Improve inventory planning

Optimize marketing campaigns

Reduce financial risk

🎯 Objectives

Perform Exploratory Data Analysis (EDA)

Apply Time Series Regression techniques

Train ML models (Linear Regression, Random Forest, XGBoost)

Evaluate model performance (MAE, RMSE, R² Score)

Deploy prediction system
